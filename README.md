### PixelParse

A Chrome Extension which converts Charts/Plots to Datasets.

### Strengths

- Performs best on:
    - bar charts
    - line charts
    - scatter plots

  ### Weaknesses

- Performs relatively weakly on:
  - Struggles with complex chart types like:
      - Sankey diagrams
      - treemaps
      - Pie charts
      - Radar Charts
      - Color Sensitive Plots
